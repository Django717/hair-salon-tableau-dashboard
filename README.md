# 📊 Hair Salon Performance Dashboard

## 📌 Project Overview
This project analyzes the performance of a hair salon using **Tableau**.  
The dashboard provides insights into revenue distribution, employee performance, service popularity, and customer inactivity patterns.

---

## 🎯 Business Questions
- Which employee generates the most revenue?
- Which service drives the highest revenue?
- Which category contributes the most to total revenue?
- How does customer inactivity evolve over time?

---

## 📊 Key Insights
- 💇‍♂️ Haircuts generate nearly 50% of total revenue
- 🏆 Evangelia leads the team by a significant margin
- 💰 Men Haircut is the top-performing service
- 📉 Customer inactivity increases significantly after 12 months

---

## 🗂 Dataset
The dataset includes:
- Employee Name  
- Service Name  
- Category  
- Gross Amount  
- Customer Segments (Inactive customers)

---

## 📈 Dashboard Features
- Revenue by employee
- Revenue by category
- Top services by revenue
- Customer inactivity segmentation

---
